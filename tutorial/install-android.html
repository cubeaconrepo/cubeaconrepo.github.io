Install APK

Step 1 - Open System Settings

Open navigation drawer - choose Settings.

Step 2 - Security Settings

Choose security menu on System Settings

Step 3 - Enable "Unknown Source"

Enable the switch of "Unknown sources" to allow installation of apps from sources other than the Play Store.

Step 4 - Download APK File

Download APK file from download page. Download page

Step 5 - Locate APK File

Locate and browse APK file that has been downloaded. Usually are located on SDCARD - Download folder.

Step 6 - Install APK File

Click and install the APK file.

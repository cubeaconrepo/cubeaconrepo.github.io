Install IPA

Step 1 - Download IPA File

Download IPA file from download page. Download page

Step 2 - Drag and Drop File

Drag-and-drop IPA file onto Library - Applications in iTunes. On Mac, you can just drag it onto the iTunes icon in the dock.

Step 3 - Verifiy App

Verify that the application shows up in Library - Applications. Note that it will not have its normal icon.

Step 4 - Connect Device

Connect your device.

Step 5 - Select Device

Select your device on iTunes.

Step 6 - Select App

In iTunes, choose the Application tab, select the application that you want to install.

Step 7 - Install App

Click on "Install" button. This will change to "Will Install"

Step 8 - Sync iTunes

Click on 'Apply' button on right corner. This will initiate process of app installation. You can see status on top of iTunes as well as app on device.

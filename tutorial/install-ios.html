Install IPA

Step 1 - Download IPA File

Download IPA file from download page. Download page

Step 2 - Drag and Drop File

Drag-and-drop IPA file onto Library - Applications in iTunes. On the Mac, you can just drag it onto the iTunes icon in the dock.

Step 3 - Verifiy App"

Verify that the application shows up in Library - Applications. Note that it will not have its normal icon.

Step 4 - Prepare to Sync

In iTunes, select your Device under Devices, choose the Application tab, and make sure that the new application is checked.

Step 5 - Sync iTunes

Sync your Device and try out the new app!
